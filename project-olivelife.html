Olive Life

대규모 트래픽 대응 뷰티 커머스 플랫폼

Java Redis Cache Load Testing

프로젝트 개요

기간: 2026.04 ~ 2026.04

역할: 백엔드 개발

Olive Life는 수많은 상품과 사용자가 존재하는 뷰티 커머스 환경을 모방하여, 대용량 트래픽 상황에서도 검색 및 결제가 원활히 이루어지는 백엔드 시스템을 구축한 프로젝트입니다.

주요 구현 및 기술적 의사결정

  • Multi-Level Cache 아키텍처: 상품 검색 시 DB 부하를 줄이기 위해 로컬 캐시와 글로벌 캐시(Redis)를 결합한 멀티 레벨 캐싱을 적용했습니다.
  • 분산 부하 테스트 환경 구축: nGrinder 등의 도구를 활용해 트래픽 병목 지점을 찾고 최적화를 진행했습니다.

성과 및 결과

캐시 레이어 최적화를 통해 특정 조건에서의 검색 응답 속도를 크게 개선하였으며, 대규모 사용자 접속 시에도 서버가 다운되지 않도록 병목 구간을 해결했습니다.
